# Caddyshack Agent Instructions

The canonical app instructions live in [`.instructions.md`](./.instructions.md).
Use that file as the source of truth for product language, betting rules, staking, cashout guidance, and asset handling.

For agent work in this repo:

- Prefer the existing services, types, and data files over introducing new abstractions.
- Keep changes tight to the area being edited.
- Update tests or validation only where the change surface justifies it.
